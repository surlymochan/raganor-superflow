## 10. 与 Superflow 工作流集成

### 阶段映射

| Superflow 阶段 | PRD 活动 | 产出 |
|----------------|----------|------|
| 阶段 1: 脑暴 | PRD 草稿 | `openspec/changes/<name>/prd-draft.md` |
| 阶段 2: 计划 | PRD 定稿 | `openspec/specs/domain/<domain>/prd-<name>.md` |
| 阶段 3: 执行 | PRD 验收验证 | 验收报告 |
| 阶段 4: 归档 | PRD 版本更新 | 归档 PRD + 版本历史 |

### 可追溯性

```
用户故事 → 功能需求 → 任务清单 → 代码提交 → 测试用例
   ↓
验收标准 ←───────────────────────────────┘
```
