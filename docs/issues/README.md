# Superflow Issues 说明

**最后更新**: 2026-03-22

---

## 概述

本目录为 **Superflow 项目 Issues 说明**。

Issues 用于追踪：
- Bug 报告
- 功能请求
- 文档改进
- 规范更新

---

## Issue 模板

### Bug 报告

```markdown
## 问题描述

清晰简洁地描述问题。

## 复现步骤

1. 步骤 1
2. 步骤 2
3. 预期行为
4. 实际行为

## 环境

- 平台：Claude Code / OpenCode / Cursor
- 版本：x.x.x

## 附加上下文

添加关于问题的任何其他上下文。
```

### 功能请求

```markdown
## 功能描述

清晰简洁地描述你想要的功能。

## 问题关联

这个功能请求是否与某个已知问题相关？

## 替代方案

描述你考虑过的替代方案。

## 附加上下文

添加关于功能请求的任何其他上下文。
```

---

## Issue 标签

| 标签 | 说明 |
|------|------|
| `bug` | 错误修复 |
| `enhancement` | 功能增强 |
| `documentation` | 文档改进 |
| `spec` | 规范更新 |
| `workflow` | 工作流相关 |
| `ci/cd` | CI/CD 相关 |
| `help wanted` | 需要帮助 |
| `good first issue` | 适合新手 |

---

## Issue 处理流程

```
1. 提交 Issue
       ↓
2. 团队审查（打标签、分配）
       ↓
3. 开发处理
       ↓
4. PR 提交
       ↓
5. CI 检查 + Code Review
       ↓
6. 合并 + 关闭
```
